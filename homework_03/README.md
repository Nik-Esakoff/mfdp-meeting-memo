# Домашнее задание №3. Бенчмаркинг продукта

Целевой продукт: локальный ВКС-парсер для наполнения базы данных корпоративных ассистентов.

Главное уточнение по контексту:
- проект рассматривается для государственной компании;
- есть ограничения на использование стороннего софта и внешних SaaS-сервисов;
- обязательное требование — обработка записей, транскриптов и фактов во внутреннем контуре.

Файлы:
- `mfdp_hw3_benchmark_table_revised.xlsx` — таблица для загрузки в Google Sheets.
- `benchmark_table.csv` — основной лист таблицы в CSV-формате.
- `mfdp_hw3_benchmark_presentation_revised.pptx` — презентация с выводами.
